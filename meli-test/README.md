# MELI API
Backend from Frontend
## Version: 1.0.0

### START SERVER 
```npm run start```

SERVER RUN IN PORT **_9010_**

### RUN TEST
```npm run test```

### CURLS SERVICE SEARCH
```curl --location --request GET 'http://localhost:9010/api/items?q=vasos'```


### CURLS SERVICE ITEM
```curl --location --request GET 'http://localhost:9010/api/items/MLA865411635'```
